## 断点续传与后台下载实现
Signal-Desktop通过`AttachmentDownloadManager`类实现了断点续传和后台下载功能。该类继承自`JobManager`，负责管理所有附件下载任务的生命周期。

```mermaid
sequenceDiagram
participant Client as "客户端"
participant Queue as "下载队列"
participant Manager as "下载管理器"
participant JobManager as "任务管理器"
participant Storage as "存储系统"
Client->>Queue : 添加消息到下载队列
Queue->>Manager : 触发flushAttachmentDownloadQueue
Manager->>JobManager : 获取待处理任务
JobManager->>Storage : 从数据库读取任务
Storage-->>JobManager : 返回任务列表
JobManager->>Manager : 启动下载任务
Manager->>Client : 下载附件
alt 下载成功
Manager->>Storage : 更新任务状态
Storage-->>Manager : 确认更新
Manager-->>Client : 通知下载完成
else 下载失败
Manager->>JobManager : 标记为重试
JobManager->>Manager : 计算重试时间
Manager->>Storage : 更新重试时间
end
```
